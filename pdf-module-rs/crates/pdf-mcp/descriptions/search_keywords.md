在 PDF 文件中搜索关键词，返回匹配位置及上下文。

## 适用场景

- 文档内容检索
- 关键信息定位
- 合同/法律文档条款查找

## 搜索算法

- 使用正则表达式匹配，支持特殊字符转义
- 支持大小写敏感/不敏感模式
- 返回每个匹配的上下文（前后 N 个字符）

## 返回数据结构

```json
{
  "keywords": ["合同", "甲方"],
  "total_matches": 15,
  "pages_with_matches": [1, 3, 5],
  "matches": [
    {
      "keyword": "合同",
      "page_number": 1,
      "text": "...本合同由甲方与乙方...",
      "bbox": [x0, y0, x1, y1],
      "start_index": 10,
      "end_index": 12,
      "confidence": 1.0
    }
  ]
}
```

## 性能特性

- 先提取结构化数据，再执行搜索
- 正则表达式会被缓存（10-100x 性能提升）
- 大文件搜索时间：~100ms/MB

## 使用建议

- `context_length` 建议 50-100 字符，太大会影响可读性
- 多关键词搜索时，结果按出现顺序排列
