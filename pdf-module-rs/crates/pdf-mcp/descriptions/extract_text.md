从 PDF 文件中提取纯文本内容。

## 适用场景

- 需要快速获取文档全文内容
- 文本分析、关键词提取的前置步骤
- 不需要保留格式和位置信息

## 提取引擎说明

- `lopdf`（默认）：布局感知引擎，适合复杂排版文档，准确度高
- `pdf-extract`：快速引擎，适合简单文档（<5页），速度最快
- `pdfium`：Chrome PDF 引擎，兼容性最强，支持特殊编码（CIDFont/Type3）

## 智能路由

不指定 adapter 时，系统会根据文档特征自动选择最优引擎：

- 小文档（≤5页）且无复杂布局 → pdf-extract
- 特殊编码 → pdfium
- 默认 → lopdf

## 性能特性

- 小文件（<1MB）：~50ms
- 中等文件（1-10MB）：~200ms
- 大文件（>10MB）：~500ms+
- 结果会被缓存（TTL: 1小时）

## 限制

- 不支持加密 PDF（会返回错误）
- 不支持扫描版 PDF（图片型文档无文本）
- 最大文件大小：100MB（可配置）

## 返回格式

纯文本字符串，保留换行符，但不保留原始格式（字体、颜色等）
