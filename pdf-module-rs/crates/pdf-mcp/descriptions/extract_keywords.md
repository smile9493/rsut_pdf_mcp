自动提取 PDF 文档中的高频关键词。

## 适用场景

- 文档主题分析
- 标签自动生成
- 文档摘要的前置步骤

## 分词算法

- 中文：使用 Jieba 分词（结巴分词），支持自定义词典
- 英文：正则提取连续字母序列（≥2字符）
- 混合文档：自动识别并分别处理

## 过滤规则

- 最小长度：2 字符（过滤单字）
- 最大长度：20 字符（过滤长句）
- 自动过滤纯数字、标点符号

## 返回格式

```json
[
  ["合同", 25],
  ["甲方", 18],
  ["乙方", 15]
]
```

## 性能特性

- Jieba 分词器全局单例缓存（5-10x 性能提升）
- 处理速度：~10ms/MB

## 使用建议

- `top_n` 建议 10-20，太多会包含无意义词汇
- 对于技术文档，可能需要后处理过滤专业术语
